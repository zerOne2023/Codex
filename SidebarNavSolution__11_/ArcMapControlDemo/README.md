# ArcMapControlDemo 使用说明

## Demo 内容
- 左侧参数面板可切换坐标校正模式：经纬度 / 三度带 / 六度带。
- 可切换椭球体（CGCS2000 / WGS84）、中央经线、比例因子、东/北偏移。
- 图层开关：OSM、天地图、SHP、本地模拟“用户库”点位图层。
- 支持“全图”定位与天地图 Token/样式动态更新。

## 运行前准备
1. 若要演示 SHP，请在输出目录下放置 `Data/sample.shp`（可配套 `.dbf/.shx`）。
2. 若要演示天地图，请在界面中填入真实 token。

## 说明
该 demo 重点演示“项目分层 + 扩展方式 + 参数化流程”，便于在大型项目中继续扩展。
